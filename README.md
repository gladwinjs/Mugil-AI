# 🌥️ Mugil AI – Career Guidance for Students

Mugil AI is a simple and intelligent career guidance web platform designed to help students explore career paths using AI-powered suggestions and structured guidance.

Built with **HTML, CSS, and JavaScript**, and deployed using **Netlify**.

---

## 🚀 Live Demo
👉 https://mugil-minds.netlify.app/

---

## 📌 About the Project

Choosing a career is confusing for many students.  
**Mugil AI** helps simplify this process by providing:

- Career exploration guidance  
- AI-based suggestions (future enhancement)  
- Student-friendly UI  
- Simple and fast web experience  
- Tamil-inspired branding with modern design  

---

## 🎯 Features

- 🎓 Career guidance landing page  
- 🌐 Responsive web design (mobile + desktop)  
- ⚡ Fast loading static website  
- 💡 Clean UI for students  
- 🌥️ Tamil-inspired branding “Mugil” (Cloud = guidance & clarity)  

---

## 🛠️ Tech Stack

- HTML5
